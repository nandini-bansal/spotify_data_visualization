#### Learn Pandas and Matplotlib in Python by visualizing your listening data into graphs of different kinds.   
## Let's Start   
To write your code, you can install either spyder (I used spyder for mine since it has pre-downloaded libraries and saves you the hustle), jupyter or any other python compiler. Make sure the libraries are also installed with it.   
You can check out the code from the uploaded python file and learn from the tutorial. It would be highly beneficial if you make one graph from the tutorial and try the rest on your own.    
For your music listening history on Spotify, visit https://www.spotify.com/us/account/privacy/ and request your data. Spotify will then prepare your data and send it to you within 5 days. I got mine on the 4th day.  


## Requirements
1. Python compiler (Spyder, Jupyter)
2. Spotify Data History
3. Pandas, Matplotlib
4. A will to learn Python
